# Power BI Beginner Dashboard – Sales Data Analysis

## Overview
This is a beginner-level Power BI dashboard built to practice the end-to-end
Power BI workflow using a sample sales dataset.

The focus of this project is on data cleaning, basic DAX calculations,
and designing clear, readable dashboards.

## Tools Used
- Power BI – dashboard creation and visualization
- Power Query – data cleaning and transformation
- DAX – basic measures and calculated fields
- Excel – initial data source

## What Was Done
- Cleaned and transformed raw sales data using Power Query
- Created basic DAX measures such as total sales and average values
- Designed bar charts, cards, and tables for analysis
- Added slicers and filters for interactivity
- Applied simple layout and formatting principles
